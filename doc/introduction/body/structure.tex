\section{程序系统的结构}
本系统采用经典的三层架构（C/S 架构的变体），即表示层、业务逻辑层和数据访问层。
\begin{itemize}
    \item \textbf{表示层 (UI)}: 负责用户界面的展示和交互，采用Java Swing构建。
    \item \textbf{业务逻辑层 (BLL)}: 处理系统的核心业务逻辑，如用户认证、消息转发等。
    \item \textbf{数据访问层 (DAL)}: 负责与数据库进行交互，实现数据的持久化存储（CRUD操作）。
\end{itemize}
